Firefly, the first social media feedback loop.

Firefly is an integrated social media feedback system of local POIs. Using a combination data mining and crowd sourcing, Firefly sorts multi-social platform data by sentiment and returns the ultimatum rating of the POI in consideration. These returns then become lights on a map interface. The more popular the POI, the brighter the light, hence the idea behind the name"fireflies".
